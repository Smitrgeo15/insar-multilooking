What is multilooking?

In InSAR, every pixel's phase is wrapped noise on top of the signal. Multilooking spatially averages complex interferometric values over L neighboring pixels to reduce phase noise by a factor of ~√L. The multilooked phase is the argument of the complex mean:

φ̂k = arg[ (1/L) Σ PM,i · PS,i* ]

This is the Maximum Likelihood estimator of the interferometric phase for distributed scatterers (Rodriguez & Martin, 1992).

Why adaptive multilooking?

Standard boxcar averaging assumes all neighbors have identical statistics. In reality, different land-cover types (field / road / building) have different scattering properties. Including heterogeneous pixels biases the estimate. Adaptive multilooking first identifies Statistically Homogeneous Pixels (SHP / brotherhood) using statistical tests (KS, AD, or parametric tests), then averages only those.

Ωp = { q : H₀ not rejected }

Phase variance vs. coherence

The Cramér-Rao lower bound for phase variance after L-looks is:

σφ2 ≥ (1 − |γ|²) / (2L|γ|²)

Where |γ| ∈ [0,1] is the coherence. High coherence + more looks → lower phase variance → better deformation estimate. This tool lets you explore that trade-off directly.
